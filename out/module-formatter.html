(static) formatItems(items, rawFormat, targetFormat) → {*}

Description:
  • 根据rawFormat，targetFormat 格式化数据并且返回新的数组
Source:
  • formatter.js, line 17
Example
const test = [ { name:'a',id:1 },{ name:'b',id:2 } ]
const result = formatItems(test)
// result => [ { name:'a',id:1,label:'a',value:1 },{ name:'b',id:2,label:'b',value:2 } ]
